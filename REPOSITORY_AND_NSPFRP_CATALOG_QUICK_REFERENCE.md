## üîÑ Catalog Maintenance System

### Maintenance Operations (Performed on Each Snap)

On each catalog snap, the system automatically performs:

1. **Resynthesize** - Rebuilds catalog from source, merges with existing
2. **Organize** - Organizes by category, octave, and status
3. **Tune** - Spectrum analysis and optimization
4. **Prevent Duplicates** - Removes duplicates based on ID + version

### Maintenance Flow

```
Catalog Snap Triggered
    ‚Üì
[RESYNTHESIZE] Rebuild from source
    ‚Üì
[PREVENT DUPLICATES] Remove duplicates
    ‚Üì
[TUNE] Spectrum analysis & optimization
    ‚Üì
[ORGANIZE] By category, octave, status
    ‚Üì
Maintained Catalog Ready
```
